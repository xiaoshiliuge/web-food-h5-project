

# Web Food H5

简介中文 

### 一、介绍

技术栈：react + node + ahooks + mock + axios + antd mobile + swiper  

### 二、快速上手

### 1、获取项目
通过`git clone`获取项目
```
git clone https://github.com/xiaoshiliuge/web-food-h5-project
```
### 2、开始安装
```
cd client
npm i

cd server
npm i
```
* `client` 是前端，`server`是服务端

### 3、运行项目
```
cd client
npm start

cd server
nodemon index.js
```
